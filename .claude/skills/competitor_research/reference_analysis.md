# Competitor Research — Analysis Framework

## Competitor Summary Format

```
COMPETITOR SUMMARY
─────────────────────────────────────────────────────────

COMPETITORS IDENTIFIED

| Competitor | Target User | Core Value Prop | Top Features | Pricing | Weakness |
|---|---|---|---|---|---|
| [Name] | [User] | [Prop] | [Features] | [Model] | [Gap] |
| ... | | | | | |

COMMON FEATURES (what all/most competitors offer)
- [Feature 1]
- [Feature 2]
- [Feature 3]

GAPS (what few or none do well)
- [Gap 1]
- [Gap 2]

DIFFERENTIATION OPPORTUNITY
[One paragraph: what your product can do differently based on the gaps found]

MARKET SIGNALS
- [Signal 1]
- [Signal 2]
─────────────────────────────────────────────────────────
```

---

## Gap Analysis Framework

Use these lenses to identify gaps:

**User segment gaps**
Is there a specific user type that all competitors underserve? (e.g., everyone targets enterprise, nobody targets indie shops)

**Feature gaps**
Is there a high-demand feature that no competitor does well?

**UX gaps**
Are competitors overcomplicated? Is there a simpler path to the same value?

**Price gaps**
Is the market overpriced or lacking a free tier that would unlock adoption?

**Platform gaps**
Is everyone a web app when users want mobile, or vice versa?

**Integration gaps**
Does your user need this to work with a tool no competitor integrates with?

## Example Analysis — Coffee Shop Loyalty

```
COMPETITOR SUMMARY
─────────────────────────────────────────────────────────

COMPETITORS IDENTIFIED

| Competitor   | Target User           | Core Value Prop         | Top Features                        | Pricing         | Weakness                        |
|---|---|---|---|---|---|
| Square Loyalty | SMB retail/café     | Full POS loyalty add-on | Points, rewards, analytics          | $45/mo add-on   | Requires Square POS              |
| Stamp Me      | Any small business   | Digital stamp cards     | Stamp cards, push notifications     | Free–$59/mo     | Generic, no café focus           |
| Yotpo         | eCommerce brands     | Reviews + loyalty       | Points, referrals, VIP tiers        | $199/mo+        | Built for eCommerce, not cafés   |
| Loopy Loyalty | Small retailers      | Digital punch cards     | Stamp cards, Apple Wallet           | $25/mo          | Outdated UI, no analytics        |

COMMON FEATURES
- Digital punch/stamp cards
- Push notification campaigns
- Basic customer dashboard

GAPS
- No product built specifically for independent coffee shops
- None offer easy menu-integrated rewards (e.g., "buy 5 lattes, get one free")
- Weak analytics for solo operators without dedicated staff

DIFFERENTIATION OPPORTUNITY
Build a mobile-first loyalty app designed around the coffee shop workflow:
check-in at point of sale, drink-specific rewards, and a simple 2-tap
owner experience for a solo operator who has no time for dashboards.

MARKET SIGNALS
- Square Loyalty's POS lock-in creates strong resentment among indie shops
- "paper punch card" is still #1 result for "coffee shop loyalty" — market is unsophisticated
─────────────────────────────────────────────────────────
```
